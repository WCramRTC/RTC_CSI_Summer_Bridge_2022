Coding Question: Fizz Buzz

You need to write a program that prints the numbers from 1 to 100 such that:

  • If the number is a multiple of 3, you need to print "Fizz" instead of that number.
  • If the number is a multiple of 5, you need to print "Buzz" instead of that number.
  • If the number is a multiple of both 3 and 5, you need to print "FizzBuzz" instead of that number.

How to Complete the FizzBuzz Challenge in 5 Programming Languages

C# - Credit To Denis for solving


for(int i = 0; i < 20; i++) {

// If number is div by 3

 if(i % 3 == 0 && i % 5 == 0) {
  Console.WriteLine("FizzBuzz");
 }
 else if ( i % 3 == 0) {
  Console.WriteLine("Fizz");
 }
 else if ( i % 5 == 0 ) {
  Console.WriteLine("Buzz");
 }
 else {
  Console.WriteLine(i);
 }
}

Console.WriteLine();
